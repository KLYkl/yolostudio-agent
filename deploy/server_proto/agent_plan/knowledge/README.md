# 知识库来源与防污染规则

当前训练知识库默认分三层理解：

- `official`：官方/通用种子规则，可直接用于默认建议
- `workflow`：当前项目系统与流程规则，可直接用于默认建议
- `case`：真实训练案例，默认不自动参与建议，需显式启用或人工确认后再用
- `test`：测试/回归用知识，默认禁止参与正式建议

## 当前默认策略

- 默认只加载 `official + workflow`
- 默认屏蔽 `test`
- 默认不自动启用 `case`
- 测试结果不得直接提升为训练经验

## case 层准入条件

只有满足以下条件的内容，后续才允许进入 `case`：

1. 来自真实训练，而不是 mock / smoke / regression
2. 有真实数据集、真实参数、真实结果
3. 最好能说明“改了什么 -> 结果如何变化”
4. 经过人工确认，不自动写回

## 当前阶段定位

当前 Phase 1 属于：

- 官方规则库
- 系统/流程知识库

不属于：

- 自动学习经验库
- 真实训练案例库
